# 🗳️ Voting Machine RTL Design

## 📌 Project Overview
This project implements a digital voting machine using RTL design. It includes simulation, synthesis, and schematic verification.

## 🛠️ Tools Used
-  Vivado (Simulation)
- Xilinx (Synthesis)
- Verilog / SystemVerilog

## ⚙️ Features
- Multiple candidate voting system
- Vote counting logic
- Reset functionality
- Debounce handling

## 📊 Simulation Results
Waveforms verify correct vote counting behavior.

## 🔧 Synthesis
Schematic generated and verified for hardware implementation.

## 📁 Folder Structure
- rtl → Design files
- testbench → Testbench code
- simulation → Waveforms
- synthesis → Schematic diagrams
